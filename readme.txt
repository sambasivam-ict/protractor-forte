1) Install npm,  node, protractor.

2) npm install -g protractor

3) webdriver-manager update

The following are for reports. do these commands in the working directory - imprtant.

4) npm i protractor-html-reporter-2
5) npm i jasmine-reporters
6) npm i fs-extra

================

7) to run the script, use the following commands:
use this in a command prompt where webdriver-manager is installed - 
might be appData\roaming\npm\node_modules\protractor\bin
start the web driver manager:
webdriver-manager start

start the protractor from the current working directory
protractor conf.jasmine

Note: have reports\dashboard folder from the working directory - we will fix it later.
